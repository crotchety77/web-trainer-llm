Логика работы ассистента разделяется на режимы в зависимости от типа пользовательского запроса и контекста выполнения образовательного процесса.
Ограничение: Для работы ИИ любому из акторов необходимо предварительно настроить API-ключ и Folder ID в своем профиле (согласно UC 1.5 и 1.6), иначе система вернет ошибку 503.

В системе нет маршрутов, которые позволяли бы обычному пользователю сменить свою роль на «Администратор».
Единственный способ назначения: Получить права администратора можно исключительно через прямой SQL-запрос к базе данных (например: UPDATE users SET role = 'admin' WHERE email = '...').

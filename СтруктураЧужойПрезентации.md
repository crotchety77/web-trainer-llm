# 2 Актуальность работы
Объём данных в ИТ быстро растёт — найти актуальную и достоверную информацию становится всё сложнее.Стандартные поисковые системы не всегда дают точный результат, особенно для новых пользователей.Возникает потребность в удобных инструментах, упрощающих поиск технической информации.

# 3 Цель и задачи
Целью выпускной квалификационной работы является разработка функционального и удобного в использовании чат-бота агрегатора, способствующего упрощению поиска технической информации и повышению эффективности работы пользователей в сфере информационных технологий.

# 4 Цель и задачи
В процессе разработки информационного чат-бота агрегатора предполагается решить следующие задачи:
1. Разработать удобный пользовательский интерфейс взаимодействия в Телеграм.
2. Настроить корректную обработку запросов пользователей.
3. Реализовать возможность поиска информации по техническим темам.
4. Обеспечить интеграцию с АРІ внешних поисковых систем и ИИ-моделей.
5. Реализовать генерацию и агрегирование справочной информации и фрагментов кода на основе запроса.
6. Обеспечить логирование сессий для повышения надёжности.

# 5 Практическая значимость работы
Практическая значимость работы заключается следующем.
1. Разработка бота способствует упрощению поиска актуальной технической информации и повышению эффективности работы пользователей в сфере информационных технологий.
2. Применение бота позволяет экономить время и другие.. ресурсы при ведении проектной и управленческой деятельности для широкого круга лиц, включая крупные и небольшие предприятия.
3. Способствует приобретению практических знаний и навыков в области проектирования и разработки интеллектуальных программных систем.

# 6 Используемые технологии
Набор технологий для разработки информационного чат-бота представляется следующим:
Язык программирования: Python.
Платформа для внедрения: Телеграмм.
Методы сбора данных: ВeautifulSoup + API поисковых систем.
Метод генерации ответов: GigaChat.

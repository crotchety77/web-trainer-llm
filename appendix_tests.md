# ПРИЛОЖЕНИЕ Х
## Каталог тест-кейсов модульного тестирования

В приложении представлены тест-кейсы модульного тестирования, отражающие основные классы входных данных, обработку исключительных ситуаций и анализ граничных значений. Формирование тест-кейсов выполнялось с использованием методов эквивалентного разбиения и анализа граничных условий. Нумерация тест-кейсов сохранена в соответствии с внутренним каталогом автоматизированных тестов проекта.

Таблицы приложения отражают структуру тестового покрытия основных компонентов платформы.

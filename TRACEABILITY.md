# Матрица трассировки: Use Cases -> API Routes

| № | Вариант использования | Актор | Реализация в системе (API Route) |
|---|----------------------|-------|---------------------------------|
| **UC-1** | **Регистрация и авторизация** | Все | |
| UC-1.1 | Регистрация нового аккаунта | Гость | `POST /api/auth/register` |
| UC-1.2 | Вход в систему (авторизация) | Гость | `POST /api/auth/login` |
| UC-1.3 | Получение данных текущей сессии | Все | `GET /api/auth/me` |
| UC-1.4 | Смена пароля и настроек | Все | `PUT /api/auth/me/password` |
| **UC-2** | **Просмотр каталога курсов** | Все | |
| UC-2.1 | Получение списка всех курсов | Студент, Гость | `GET /api/courses` |
| UC-2.2 | Просмотр детальной информации о курсе | Студент, Гость | `GET /api/courses/:id` |
| **UC-3** | **Запись на курс** | Студент | `POST /api/courses/:courseId/enroll` |
| **UC-4** | **Прохождение обучения** | Студент | |
| UC-4.1 | Получение структуры урока и блоков | Студент | `GET /api/lessons/:id` |
| UC-4.2 | Прохождение теоретического блока | Студент | `POST /api/blocks/:id/complete` |
| UC-4.3 | Список курсов студента (прогресс) | Студент | `GET /api/student/courses` |
| **UC-5** | **Выполнение теста (опрос)** | Студент | `POST /api/blocks/:id/submit` |
| **UC-6** | **Отправка кода (практика)** | Студент | `POST /api/blocks/:id/submissions` |
| **UC-7** | **Просмотр результатов** | Студент | Ответ от UC-5/UC-6 (поле `tests_result`) |

| **UC-8** | **ИИ-ассистент** | Студент, Автор | `POST /api/ai/chat` (Yandex GPT) |
| **UC-9** | **Создание и настройка курса** | Автор | |
| UC-9.1 | Создание нового курса | Автор | `POST /api/courses` |
| UC-9.2 | Редактирование метаданных курса | Автор | `PATCH /api/courses/:id` |
| UC-9.3 | Список курсов автора | Автор | `GET /api/my/courses` |
| **UC-10** | **Управление уроками и блоками** | Автор | |
| UC-10.1| Создание/изменение/удаление уроков | Автор | `POST /api/courses/:id/lessons`, `PATCH /api/lessons/:id`, `DELETE /api/lessons/:id` |
| UC-10.2| Создание/изменение/удаление блоков | Автор | `POST /api/lessons/:id/blocks`, `PATCH /api/blocks/:id`, `DELETE /api/blocks/:id` |
| **UC-11** | **Настройка заданий** | Автор | |
| UC-11.1| Настройка тестов и кейсов | Автор | `PATCH /api/blocks/:id` (поле `quiz_data`) |
| **UC-12** | **Тестирование задания (sandbox)** | Автор | `POST /api/run-tests` (Piston API) |
| **UC-13** | **Публикация курса** | Автор | `PATCH /api/courses/:id` (поле `is_published`) |
| **UC-14** | **Администрирование** | Админ | |
| UC-14.1| Просмотр списка пользователей | Админ | `GET /api/admin/users` |
| UC-14.2| Удаление пользователя и данных | Админ | `DELETE /api/admin/users/:id` |
